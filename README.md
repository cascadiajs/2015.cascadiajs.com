# The website for CascadiaJS 2015

[2015.cascadiajs.com](http://2015.cascadiajs.com)

Powered by [surge](http://surge.sh/).

To check out:

```sh
git clone https://github.com/cascadiajs/2015.cascadiajs.com.git
git checkout harp
```

To get running:

```sh
npm install
npm start
# Visit http://localhost:9000
```

To deploy:

```sh
npm run deploy
```
